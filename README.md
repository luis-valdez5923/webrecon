# Web Recon Tool

This is a command-line web reconnaissance tool that supports various fuzzing modes to discover information about a target website.

## Features

- Subdomains fuzzing
- Directory fuzzing
- Virtual host (vhost) fuzzing
- Extension fuzzing
- Page fuzzing
